# API Endpoints Test Plan

This document outlines the API endpoints for the CareerConnect backend and provides a structure for testing them.

## User Flow:
1.  **Authentication**: A user signs up and logs in.
2.  **Profile Management**: The user updates their profile.
3.  **Assessment**: The user takes a career assessment.
4.  **Career Exploration**: The user explores career recommendations and searches for careers.
5.  **Mentor Interaction**: The user finds and books a session with a mentor.
6.  **AI Chat**: The user interacts with the AI career counselor.
7.  **Sign Off**: The user logs out (client-side token removal).
